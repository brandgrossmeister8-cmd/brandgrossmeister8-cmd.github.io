Рита хочет создать скиллс/шаблон для разработки бизнес-маркетинговых игр по аналогии с "Маркетинговый заезд".

**Why:** У неё есть технология системного продвижения и несколько методологий. Хочет создавать аналогичные игры быстро.

**How to apply:** Когда вернётся к этому вопросу — предложить начать с Пути 1 (шаблон + конфиг), затем при необходимости перейти к Пути 2 (конструктор с админкой). Текущий движок уже параметризован через stages.ts.
