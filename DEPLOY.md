# 🚀 Быстрый старт - Обновление игры

## Для быстрого обновления игры используйте:

```bash
./deploy.sh "Ваше описание изменений"
```

**Примеры:**

```bash
# С описанием
./deploy.sh "Исправлена ошибка с таймером"

# Без описания (будет автоматическая дата)
./deploy.sh
```

---

## Что делает скрипт:

1. ✅ Проверяет изменения в коде
2. ✅ Создаёт git коммит
3. ✅ Отправляет на GitHub
4. ✅ Собирает проект (`npm run build`)
5. ✅ Публикует на GitHub Pages
6. ✅ Показывает ссылку на игру

---

## После публикации:

- Игра доступна по адресу: **https://brandgrossmeister8-cmd.github.io/race-to-insight/**
- Изменения применяются через **1-2 минуты**
- Если не видите изменений - обновите страницу (`Cmd + Shift + R`)

---

## Если скрипт не запускается:

```bash
chmod +x deploy.sh
```

---

## Альтернативный способ (вручную):

```bash
# 1. Коммит
git add .
git commit -m "Описание"
git push

# 2. Публикация
npm run build
npx gh-pages -d dist
```
